# TwitterStyleLaravel
## 開発環境
Windows10 Pro

Laravel Framework 6.20.4

php7.4

10.4.14-MariaDB

## 使い方
プロジェクトをクローンする
```
git clone https://github.com/ue8d/TwitterStyleLaravel.git
```

必要なものをダウンロードする
```
npm install
npm run dev
```

ストレージをリンクさせる
```
php artisan storage:link
```

サーバを起動する
```
php artisan serve
```
